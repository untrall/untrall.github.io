Chapter 3 Streams

ifstream

ifstream is only a type not a variable

eg:

ifstream myStream(“myFile.txt”);

another version:

ifstream myStream;
myStream.open(“myFile.txt”);

有时候，读取文件的时候文件无法打开，会读取到一些没用的东西，所以可以加入.is_open()来判断是否成功打开：

ifstream input(“myfile.txt”);

if(!input.is_open())
　　cerr<<”Couldn’t open the file myfile.txt”<<endl;

cerr is designed for error reportingand is sometimes handled differently by the os.
ofstream

ofstream myStream(“myFile.txt”); //Write to myFile.txt
如果ofstream的文件不存在，就会创一个新的，如果已经存在了，就会覆盖掉所有的内容，所以用的时候要小心！！！
ifstream input(myString.c_str()); //Open the filename stored in myString
因为streams 库是c++中比较古老的一个库，open functions on the ifstream and ofstream classes predate the string type，所以想把”string”传给 open的时候，需要其是一个C-style string，而不是”string”（maybe 是个class） 可以用.c_str（）来进行转换。

eg:

ifstream input(myString.c_str()); //Open the filename stored in myString

setw(20) //set width for cout,default pad whitespaces on the left
setfill() //用来填补setw()多余的部分
stream.fail() //用来check if a stream is in an error state
stream.clean() //用来bring stream out of its error state

An Alternative: getline()

getline() accepts two parameters

1.a stream to read from
2.a string to write to.

code:

string myStr;
getline(cin, myStr);

concise one:

int main()
{
ifstream capitals(“world-capitals.txt”)
if (!capitals.is_open()) {
cerr << “Cannot find the file world-capitals.txt” << endl;
return -1;
}

string capital, country;
while (getline(capitals, capital) && getline(capitals, country))
cout << capital << “ is the capital of “ << country << endl;
}

Lecture 5 ：Advanced Containers

Multimap: 一个key可以有两个不同的值，没有[ ]操作符， add elements by calling insert on a key value std::pair
e.g.:
multimap<int,int> myMMap;
myMMap.insert(make_pair(3,3));
myMMap.insert({3,3}); // shorter syntax
cout<< myMMap.count(3) << endl; // prints 2

Iterators provide a guaranteed interface!

Four essential iterator operations:

Create iterator

std::set<int>::iterator iter = mySet.begin();

Dereference iterator to read value currently pointed to

int val = *iter;

Advance iterator

iter++; or ++iter;

Compare against another iterator(especially .end() iterator

if(iter == mySet.end()) return;

Iterators let us view a non_linear collection in a linear manner.

one iterators example: (int can be replaced by list or vector)
int numOccurrences (set<int>& cont, int elemToCount){
    int counter = 0;
    set<int>::iterator iter;
    for(iter = cont.begin(); iter != cont.end(); ++iter){
        if(*iter == elemToCount)
            ++counter;
    }
    return counter;
}

map<int, int> m;
map<int, int>::iterator i = m.begin();
map<int, int>::iterator end = m.end();
while(i != end){
    cout << (*i).first << (*i),second << endl;   //需要不仅要*i 还需要(*i).first (*i).second 因为map有两个元素
}

    // we can use iterators in algorithms like sort to sort a vecotr
